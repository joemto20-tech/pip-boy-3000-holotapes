# Pip-Wordle

### Info

**Author(s):**

- [@amogus-67](https://github.com/amogus-67)

### Description

A Wordle clone for the Pip-Boy 3000.

### Instructions

Guess the 5-letter word in 6 tries. Tiles and on-screen keys are color-coded to
show correct letters, letters in the wrong spot, and letters not in the word.

### Controls

| Input       | Action                          |
| ----------- | ------------------------------- |
| Knob 1      | Move keyboard cursor up/down    |
| Knob 2      | Move keyboard cursor left/right |
| Wheel click | Select highlighted key          |
