### Description

A Fallout-themed take on the classic Tic-Tac-Toe. Challenge a friend in 2-Player
mode or take on the CPU.

### Controls

- Left knob scroll: Move selection vertically
- Left knob press: Confirm selection
- Right knob scroll: Move selection horizontally

### Instructions

1. Choose **2-Player** or **VS CPU** at the start screen.
2. Players alternate placing their marker (X or O) on the 3×3 grid.
3. First to place three in a row, horizontal, vertical, or diagonal, wins.
4. If all nine cells are filled with no winner, the game ends in a draw.
