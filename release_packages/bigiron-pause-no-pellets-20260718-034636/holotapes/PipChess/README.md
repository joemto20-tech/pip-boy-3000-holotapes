# Pip Chess

### Info

**Author(s):**

- [@OrionMissesRust](https://github.com/OrionMissesRust)

### Description

Chess on the Pip-Boy!

### Controls

Left scroll wheel: Scroll up and down to control vertical movements.

Right scroll wheel: Scroll left and right to control horizontal movements.

### Setup \& Installation

Add `APP.JS` to `/HOLO` and add `PIPCHESS.info` to `/APPINFO`
