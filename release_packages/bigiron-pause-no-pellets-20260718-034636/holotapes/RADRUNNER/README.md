# RAD RUNNER

### Info

**Author(s):**

- [@SukeMobile](https://github.com/SukeMobile)

### Description

An endless three-lane wasteland runner for the Wand Company Pip-Boy 3000. Bomb
down a cracked desert highway, dodge the wrecks, leap the radroaches, duck the
busted signs, and gun down the ghouls. Score is distance survived.

### Controls

Right wheel — rotate: switch lanes (left / centre / right)

Left wheel — up: jump (clear a radroach) Left wheel — down: duck (under an
Atomic Cola sign)

Left wheel — click: shoot (drops the nearest ghoul or radroach in your lane)

Either wheel — click on the title / game-over screen: start or restart Wrecked
cars can't be jumped, ducked, or shot — change lanes to dodge them.

### Gameplay

Speed and obstacle density ramp up the longer you survive. At higher speeds,
full-width walls of radroaches or signs appear, forcing a jump or duck rather
than a lane change. The game runs on real elapsed time, so it plays at a
consistent speed regardless of frame rate. Tested Firmware: V1.1.3 (2v20.260)
