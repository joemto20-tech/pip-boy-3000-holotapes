# Poker

### Info

**Author(s):**

-[@Theeohn](https://github.com/Theeohn)

### Description

A single-player poker game against a dealer CPU. 5 and 7 card draw game modes
with a betting system in place.

### Controls

| Input                                      | Action                                        |
| ------------------------------------------ | --------------------------------------------- |
| Left scroll wheel (knob1) up/down          | Adjust bet amount (10-chip increments) and    |
| scroll along dealt hand to select discards |
| Right scroll wheel (knob2) left/right      | Scroll along dealt hand to select discards    |
| Left scroll wheel press                    | Deal / Confirm action / Continue after result |

### Notes

- Session-only chip balance — resets to 500 on app restart.
- Idle filter suppressed during app, restored on exit.
