### Description

Crack locked RobCo terminals by guessing the correct password from a list of
corrupted pre-war entries.

### Controls

- Use the left knob to move the cursor up and down.
- Use the right knob to move the cursor left and right.
- Press the left knob button to select options.
- On the start screen, use the left or right knob to choose the challenge level.

### Instructions

Each failed attempt reveals a score, showing how many letters match the correct
password in the right position. Use logic, deduction, and luck to narrow down
the answer before your attempts run out.

### Tips

Hidden bracket bonuses start with `(`, `[`, `{`, or `<` and end with the
matching closing bracket on the same row. Move the cursor to the opening bracket
and select it to trigger a one-use bonus.

Bonus effects:

- `ATTEMPTS RESTORED` refills attempts back to four.
- `FALSE KEY REMOVED` masks one visible wrong access key.

After a security delay, the holotape starts a new round automatically.

### Note

The game may pause for a few seconds after selecting an option while it updates.
Wait until it responds again before pressing anything, or extra button presses
may trigger later.
